This folder is intended for Arduino R4 LED Matrix examples to use as a starting point.

Follow this order for the examples in this folder:
1) Basic - ON, Happy, Heart
2) Wink - Wink Animation
3) GameOfLife - Game Of Life Animation
4) TextWithArduinoGraphics - Any text you enter in the "text" field provided in the code, can be converted to a horizontally scrolling text on the LED Matrix
