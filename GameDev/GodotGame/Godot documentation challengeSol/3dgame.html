Opening the project

  1. Open Godot_v4.2.2-stable_win64 from Godot_v4.2.2-stable_win64.exe folder
  2. Click on import
  3. Enter the following path in the dialog box: /Users/Downloads/ASU-SUCCESS/GameDev/GodotGame/dino_run and click on Select Current Folder button below.
  4. Click on Import and Edit and then open the project by click on 3D game.
  5. Now the next step is to test and run the game to make sure everything is working fine. Click on the play button.

Movements in the Scene

For placing objects into the scene, movement is the scene must be known

  1. Click on 3D from the top Bar.
  2. Now Right Click on the mouse and while continuing press the following key on the keywords for movement.
    1. For Moving closer to the scene Press W.
    2. For Moving Away to the scene Press S.
    3. For Moving Left PressA.
    4. For Moving Right PressD.
    5. For Moving UP PressE.
    6. For Moving Down PressQ.

Changing Parameters

  1. Now after you run the game you may want to increase speed of object, Go to player.gd script and click on Script iconon the top of window and change the value of speed line 9 to your desired value.
  2. Now if you want to make the player jump at place, go to line 46 and uncomment it using Ctrl+/.
  3. Now save the script using Ctrl+Sand again click on play button

Changing Key Bindings

Game always requires settings to be changed, Let's change Key Bindings!!!

  1. Click on the Project from the Navabar as shown in image and then click on Project Settings
  2. Click on Input Mapunder project settings and then under action find the pencilicon
  3. Under Action you will be able to see for which action which keys are already selected. Now Click on Pencil and choose any of the key for action.
    4. Remember Key should be different for each actions.
  4. Click on OKand click on Play Button. You will be able to see the player move according to your key bindings

Customizing the Obstacle

  1. In the Scenes folder, double click on Obstacle.tscnfile and then click on 3Dfrom the top.
  2. Click on MeshInstance3D on left pane
  3. Now go the Inspectorwindow on the right hand side of the window and in Mesh choose any of the solid shape likeNew BoxMesh, New CapsuleMesh, New CylinderMesh, New SphereMesh.
  4. Now expand the Asset folder from the left and drag an image of your choice to Material Overrideand Material Overlayunder Geometry GeometryInstance3D.
  5. Now Click on CollisionShape3D and under CollisionShape3D on same shape what you chose in MeshInstance3D.
  6. Save the Obstacle using Ctrl+S.
  7. Click on Play button to the see the changes in the game.

Adding Object to the scene

  1. Double click on main.tscn and make sure that main appears on the left pane
  2. Right on Main and Click on Instantiate Child Scene and select on Obstacle from the scene.
  3. You will see your obstacle appears on the main scene in 3Dwindow and also Obstacle under main
  4. Now click on the obstacle under main and you will see three axis visible.Green will be Y-axis(position.y), Red will be x-axis(position.x),Blue will be Z-axis(position.z).
  5. Now you can drag any of the axis and place the obstacle at the desired position make sure it is within the floor.
  6. Save your scene.

Exporting to Web

  1. Click on the Project from the top and then click Export...
  2. Click on Add... besides presets and select Web
  3. Click on download and install present on the bottom of the window.
  4. Then in Preset, click on Web, under web click Export Project
  5. Create a folder and in folder save the file with name of your choice with .html.
  6. Add Server.py file to folder where you added the file.
  7. Go the folder and type cmd in the place surronded by red box in the image and press Enter
  8. In the new window typepython server.py
  9. Web browser will open now and now find your file and double click on it.
  10. Hey !!! your game started to run on web
