Link for the 2D game: https://drive.google.com/drive/folders/10bVAmfwHBt6ALaDP87-SriUQ_QTIZzQE?usp=sharing

1. Download the game from the link above.
2. Unzip the game to a folder.
3. Open UnityHub and login there using a google account.
4. Click on add a project and select the unzipped game folder.
5. Once the new game shows up, click on it to open it.
6. Click on play to start the game.
